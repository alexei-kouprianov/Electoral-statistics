Переменные, характеризующие позицию Участковой избирательной комиссии в иерархии избирательных комиссий

level :: Техническая переменная: наивысший уровень Избирательных комиссий для данных выборов
region :: Регион
oik :: Окружная избирательная комиссия
tik :: Территориальная избирательная комиссия
uik :: Участковая избирательная комиссия

Переменные, значения которых взяты из протоколов Участковых избирательных комиссий (переменные с названиями политических партий содержат ряды чисел соответствующие количеству отданных каждой партии голосов):

VOTERS :: Число избирателей, внесенных в список избирателей на момент окончания голосования
BALL.RECEIVED :: Число избирательных бюллетеней, полученных участковой избирательной комиссией
BALL.PRELIM :: Число избирательных бюллетеней, выданных избирателям, проголосовавшим досрочно
BALL.ISSUED.INSIDE_POLLING_STATION :: Число избирательных бюллетеней, выданных в помещении для голосования в день голосования
BALL.ISSUED.OUTSIDE_POLLING_STATION :: Число избирательных бюллетеней, выданных вне помещения для голосования в день голосования
BALL.CANCELLED :: Число погашенных избирательных бюллетеней
BALL.PORTABLE.BOXES :: Число избирательных бюллетеней, содержащихся в переносных ящиках для голосования
BALL.STATIONARY.BOXES :: Число избирательных бюллетеней, содержащихся в стационарных ящиках для голосования
BALL.INVALID :: Число недействительных избирательных бюллетеней
BALL.VALID :: Число действительных избирательных бюллетеней
BALL.LOST :: Число утраченных избирательных бюллетеней
BALL.NOT_RECORDED :: Число избирательных бюллетеней, не учтенных при получении
